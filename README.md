# Telegram-бот для коллективной генерации кода

Бот позволяет коллективно писать код — участники голосуют за следующую строку программы, а ИИ (через Cloudflare Llama 3.3-70B) предлагает варианты.

Каждые несколько минут бот публикует опрос с четырьмя вариантами кода. После завершения опроса побеждает строка с наибольшим количеством голосов, и она добавляется в историю. Администратор может завершить сессию, получить итоговый код и начать новый цикл.

## Стек

- python 3.10+
- python-telegram-bot 21.x
- Cloudflare AI API
- dotenv для конфигурации
- requests для HTTP-запросов

## Установка
1. Клонируйте репозиторий:
```
git clone https://github.com/ilyas13z/sber_telegram_bot_test_task
cd sber_telegram_bot_test_task
```

2. Создайте и активируйте виртуальное окружение:
```
python -m venv venv
source venv/bin/activate       # Linux / macOS
venv\Scripts\activate          # Windows
```

3. Установите зависимости:
```
pip install -r requirements.txt
```

## Настройка окружения
Создайте файл `.env` в корне проекта:
```
cp .env.example .env
```

## Запуск
```
python bot.py
```

## Логирование
Все действия записываются в `bot.log`
Ошибки, создание опросов, ответы пользователей — всё фиксируется для отладки.
